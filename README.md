# Ontologies

This repository contains resources for one or more projects, including work on misinformation analysis and other future efforts.

## Structure

- `Misinformation_Disinformation_Analysis/`
  - `Bug_Fixes/`
    - `Bug_fixes_ontology.tex` - LaTeX document describing fixes or updates to the ontology.
  - `Cyphers/`
    - `misdetect_publication_final.cypher` - Cypher script for Neo4j graph database operations related to misinformation publication detection.
  - `Ontology/`
    - `misinformation_ontology.ttl` - RDF/Turtle ontology file defining misinformation concepts.

## Purpose

The repository is intended to support the creation, correction, and analysis of ontology and project resources, including:

- ontology modeling in Turtle format
- quality and bug fix documentation in LaTeX
- graph database queries in Neo4j Cypher

## Usage

- Review the ontology in `Misinformation_Disinformation_Analysis/Ontology/misinformation_ontology.ttl`
- Apply or test graph queries with Neo4j using `Misinformation_Disinformation_Analysis/Cyphers/misdetect_publication_final.cypher`
- Read the ontology bug fix notes in `Misinformation_Disinformation_Analysis/Bug_Fixes/Bug_fixes_ontology.tex`

## Notes

This repository is a collection of project artifacts and reference resources. It is designed to grow with additional project-specific folders and files.
